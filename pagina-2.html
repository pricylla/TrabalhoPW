Instalando Python no Windows

Para instalar o python no seu sistema operacional windows, você precisa baixar o instalador. Acesse o site oficial https://www.python.org/downloads/ e clique em download. Faça o download no instalador executável do windows (32 ou 64 bits) e clique duas vezes nele para iniciar o assitente de instalação do python.

O processo de instalação é bem simples.
1. Marque a opção "add python to PATH"
2. Clique em "install now"
